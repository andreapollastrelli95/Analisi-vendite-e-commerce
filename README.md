# Analisi-vendite-e-commerce
Questo progetto analizza le performance di un e-commerce focalizzandosi sull'ottimizzazione del funnel di conversione. L'obiettivo è trasformare i dati grezzi sul comportamento degli utenti in decisioni strategiche per aumentare le vendite. Obiettivi e Risultati Raggiunti: Aumento Tasso di Conversione, Monitoraggio KPI, Performance prodotti.
